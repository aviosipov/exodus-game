# מבוא למשחק "יציאת מצרים"

## מהו המשחק?

**יציאת מצרים: הרפתקה אינטראקטיבית** (או בשמו הנוסף: "כיצד תספר: המשחק") הוא משחק ויזואל נובל (Visual Novel) אינטראקטיבי המבוסס על סיפור יציאת מצרים. המשחק מאפשר לשחקנים לחוות את המסע של בני ישראל מהמעמד המכובד במצרים, דרך השעבוד הקשה ועד לגאולה, תוך שילוב אלמנטים חינוכיים וחווייתיים.

## קהל היעד

המשחק מיועד ל:
* ילדים בגילאי יסודי-חטיבה
* משפחות
* כל מי שמעוניין לחוות את סיפור יציאת מצרים בצורה אינטראקטיבית ומרתקת

## מטרות המשחק

1. **הנגשת הסיפור** - להנגיש את סיפור יציאת מצרים בצורה חוויתית ואינטראקטיבית
2. **הדגשת מסרים מרכזיים** - להדגיש את המסרים המרכזיים של הסיפור (אמונה, חירות, התערבות אלוקית, חוסן)
3. **חוויה מהנה** - ליצור חווית משחק מהנה ומעוררת מחשבה
4. **מעורבות** - לשלב אלמנטים משחקיים (מיני-משחקים) להגברת המעורבות

## עולם המשחק והסיפור

העלילה עוקבת אחר הנרטיב המרכזי של יציאת מצרים:

1. **המעמד המכובד** - המעמד המכובד הראשוני של בני ישראל במצרים
2. **ההידרדרות** - ההידרדרות ההדרגתית לשעבוד (תחילה ב"פה-רך" ואז בעבודת פרך)
3. **הקושי והייאוש** - תיאור הקושי, הייאוש וחוסר האונים של העבדות
4. **תהליך היציאה** - תהליך היציאה ההדרגתי (מכות מצרים)
5. **היציאה הפיזית** - היציאה הפיזית ממצרים וקריעת ים סוף
6. **קבלת התורה** - קבלת התורה והמסע לארץ ישראל

המשחק מחולק לפרקים התואמים את שלבי הסיפור (כבוד -> שעבוד רך -> שעבוד קשה -> יציאה -> גאולה).

## דמויות מפתח

* **אוהד** - עברי המייצג את המעמד הגבוה והנכבד בתחילת הדרך
* **יששכר** - תלמיד חכם עברי, זוכה לכבוד בתחילה
* **אוזיריס** - פועל מצרי שעובד אצל אוהד
* **אחמוס** - סוחר מצרי עשיר
* **פרעה** - השליט המצרי, מטיל הגזירות והשעבוד
* **קריין** - מדריך את השחקן ומספק הקשר סיפורי

## מכניקות משחק

### ליבה (Visual Novel)

* **הצגת הסיפור** - דרך טקסט (קריין, דיאלוגים) ותמונות רקע/דמויות
* **בחירות דיאלוג** - בנקודות מפתח, השחקן יכול לבחור את תגובת הדמות

### מיני-משחקים

המשחק משלב מספר מיני-משחקים בנקודות מפתח בסיפור:

1. **"השלם את הפסוק / הציווי"** - משחק טריוויה בסגנון "מי רוצה להיות מיליונר"
2. **"איסוף וארגון"** - משחק מבט-על פשוט לאיסוף פריטים
3. **"ניהול משימות צוות יומי"** - משחק ניהול משאבים קל
4. **"שמירה על התקווה"** - הסבה של משחק "Space Invaders"

## טכנולוגיה

המשחק פותח באמצעות:
* **פלטפורמה**: Next.js (React)
* **עיצוב**: Tailwind CSS
* **אנימציות**: Anime.js

## סיכום

משחק "יציאת מצרים: הרפתקה אינטראקטיבית" מציג את סיפור יציאת מצרים בצורה מרתקת, חינוכית ואינטראקטיבית. על ידי שילוב מכניקות Visual Novel עם מיני-משחקים מגוונים, המשחק יוצר חוויה ייחודית שמחברת את השחקנים לסיפור המכונן הזה בצורה עמוקה ומשמעותית, תוך שימת דגש על מסרים של חוסן, תקווה וגאולה.
