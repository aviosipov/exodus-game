# ניהול תמונות ונכסים

מדריך זה יסביר כיצד לנהל את התמונות והנכסים החזותיים האחרים המשמשים במשחק "יציאת מצרים". נכסים אלו כוללים תמונות רקע, תמונות דמויות, אייקונים וכל קובץ אחר שמוצג לשחקן.

## היכן נמצאים הנכסים?

כל הנכסים החזותיים של המשחק מאוחסנים בתיקייה `public`. תיקייה זו נגישה ישירות מהדפדפן, ולכן קל לקשר אליה מהקוד ומהתוכן.

### מבנה תיקיית `public`

בתוך תיקיית `public`, הנכסים מאורגנים בדרך כלל בתיקיות משנה לפי סוג או שימוש:

* **`public/characters`**: מכילה תיקיות משנה לכל דמות, ובתוכן תמונות של אותה דמות (לדוגמה, `public/characters/ohad/image.png`).
* **`public/images`**: מכילה תיקיות משנה לתמונות רקע, ממשק משתמש וכו'.
    * `public/images/adventures`: תמונות רקע להרפתקאות (לדוגמה, `public/images/adventures/days_of_honor/prosperity_1.png`).
    * `public/images/ui`: תמונות לממשק המשתמש (כפתורים, אייקונים וכו').
* **`public/audio`** (אם קיים): קבצי שמע למשחק.

## כיצד להחליף תמונה קיימת

הדרך הפשוטה ביותר לשנות תמונה במשחק היא להחליף את הקובץ הקיים בקובץ חדש:

1. **אתרו את הקובץ**: מצאו את קובץ התמונה שברצונכם להחליף בתוך תיקיית `public`. לדוגמה, אם אתם רוצים להחליף את תמונת הרקע הראשונה של "ימי הכבוד", הקובץ יהיה `public/images/adventures/days_of_honor/prosperity_1.png`.
2. **הכינו את התמונה החדשה**: וודאו שהתמונה החדשה שלכם מוכנה ובפורמט מתאים (לדוגמה, PNG, JPG, WEBP).
3. **שנו את שם הקובץ החדש**: שנו את שם הקובץ של התמונה החדשה כך שיהיה **זהה לחלוטין** לשם הקובץ המקורי שאתם רוצים להחליף.
4. **החליפו את הקובץ**: העתיקו את התמונה החדשה (עם השם הזהה) לתיקייה המתאימה בתוך `public`, והחליפו את הקובץ הקיים.
5. **בדקו במשחק**: הריצו את המשחק ובדקו שהתמונה החדשה מוצגת כראוי.

**חשוב**: שמירה על שם קובץ זהה מבטיחה שכל המקומות במשחק שמפנים לתמונה המקורית יפנו כעת לתמונה החדשה ללא צורך בשינוי קוד או קבצי תוכן.

## כיצד להוסיף תמונה חדשה

אם אתם רוצים להוסיף תמונה חדשה לגמרי למשחק (לדוגמה, רקע חדש להרפתקה חדשה או דמות חדשה):

1. **הכינו את התמונה**: וודאו שהתמונה החדשה מוכנה ובפורמט מתאים.
2. **בחרו מיקום ושם**: החליטו היכן למקם את התמונה בתוך תיקיית `public` (לדוגמה, בתיקייה קיימת כמו `public/images/adventures/new_adventure` או צרו תיקייה חדשה). בחרו שם קובץ ברור ותיאורי (לדוגמה, `new_background.png`).
3. **העתיקו את הקובץ**: העתיקו את התמונה החדשה למיקום שבחרתם בתוך תיקיית `public`.
4. **עדכנו את קבצי התוכן**: כעת עליכם לעדכן את קבצי ה-JSON הרלוונטיים (לדוגמה, קובץ הרפתקה) כדי שיפנו לתמונה החדשה. מצאו את השדה המתאים (לדוגמה, `background_img` או `character_left_img`) ושנו את הנתיב כך שיצביע על הקובץ החדש שהוספתם:

   ```json
   {
     "speaker": "מספר",
     "text": "זוהי סצנה עם רקע חדש.",
     "background_img": "/images/adventures/new_adventure/new_background.png" // נתיב לתמונה החדשה
   }
   ```

5. **בדקו במשחק**: הריצו את המשחק ובדקו שהתמונה החדשה מוצגת במקום הנכון.

## פורמטים מומלצים לתמונות

* **PNG**: מתאים לתמונות עם שקיפות (כמו דמויות ללא רקע) או גרפיקה עם קצוות חדים.
* **JPG/JPEG**: מתאים לתמונות רקע או תמונות עם הרבה צבעים ופרטים (כמו צילומים). מאפשר דחיסה טובה יותר.
* **WEBP**: פורמט מודרני המציע דחיסה טובה יותר מ-JPG ו-PNG, ותומך גם בשקיפות. מומלץ לשימוש אם אין בעיות תאימות.
* **SVG**: מתאים לאייקונים ולוגואים פשוטים. ניתן לשנות את גודלם ללא איבוד איכות.

## טיפים לניהול נכסים

### 1. שמות קבצים ברורים

השתמשו בשמות קבצים תיאוריים באנגלית כדי שיהיה קל להבין מה מכיל כל קובץ (לדוגמה, `market_stall.png`, `ohad_happy.png`).

### 2. ארגון בתיקיות

שמרו על מבנה תיקיות הגיוני בתוך `public` כדי שיהיה קל למצוא קבצים.

### 3. אופטימיזציה של גודל קבצים

תמונות גדולות מדי יכולות להאט את טעינת המשחק. השתמשו בכלים לאופטימיזציה של תמונות כדי להקטין את גודל הקבצים מבלי לפגוע משמעותית באיכות. ישנם כלים מקוונים רבים כמו [TinyPNG](https://tinypng.com/) או תוכנות עריכת תמונות המאפשרות שמירה מותאמת לרשת.

### 4. שמירה על עקביות סגנונית

נסו לשמור על סגנון ויזואלי אחיד בכל התמונות והנכסים במשחק.

## סיכום

ניהול תמונות ונכסים במשחק "יציאת מצרים" מתבצע בעיקר דרך תיקיית `public`. על ידי החלפת קבצים קיימים או הוספת קבצים חדשים ועדכון הנתיבים בקבצי התוכן, תוכלו לשנות את המראה והתחושה של המשחק ולהתאים אותו לצרכים שלכם.

זכרו להשתמש בשמות קבצים ברורים, לארגן את הקבצים בתיקיות, לבצע אופטימיזציה לגודל הקבצים ולשמור על עקביות סגנונית.
