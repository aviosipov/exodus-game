# יצירה ועריכת תוכן - סקירה כללית

מדריך זה מספק סקירה כללית של תהליך עריכת התוכן במשחק "יציאת מצרים". הוא מיועד למשתמשים ללא רקע טכני שרוצים להתאים את תוכן המשחק לצרכיהם.

## סוגי התוכן במשחק

משחק "יציאת מצרים" מורכב ממספר סוגי תוכן עיקריים:

1. **הרפתקאות** - הסיפור המרכזי, דיאלוגים ובחירות
2. **טריוויה** - שאלות ותשובות בנושאי יציאת מצרים
3. **נכסים חזותיים** - תמונות רקע, דמויות ואייקונים
4. **מיני-משחקים** - משחקים קטנים המשולבים בחוויה

## כלים נדרשים לעריכת תוכן

לעריכת תוכן במשחק, תזדקקו לכלים הבאים:

1. **עורך טקסט** - לעריכת קבצי JSON ו-MDX
   * מומלץ: [Visual Studio Code](https://code.visualstudio.com/) (חינמי)
   * אלטרנטיבות פשוטות: Notepad++ (Windows), TextEdit (Mac)

2. **עורך תמונות** (אופציונלי) - לעריכת תמונות
   * אפשרויות חינמיות: [GIMP](https://www.gimp.org/), [Paint.NET](https://www.getpaint.net/)
   * אפשרויות בתשלום: Adobe Photoshop, Affinity Photo

3. **כלי ניהול גרסאות** (אופציונלי, אך מומלץ)
   * [Git](https://git-scm.com/) - לשמירת היסטוריית השינויים

## תהליך עריכת תוכן - צעדים כלליים

### 1. הכנה

1. **גיבוי** - תמיד גבו את הקבצים המקוריים לפני עריכה
2. **תכנון** - חשבו מראש על השינויים שאתם רוצים לבצע
3. **הכרת המבנה** - הבינו את מבנה הקבצים (ראו [מבנה הפרויקט](/docs/project-structure))

### 2. עריכה

1. **פתיחת הקובץ** - פתחו את הקובץ הרלוונטי בעורך הטקסט
2. **ביצוע שינויים** - ערכו את התוכן בהתאם לצורך
3. **שמירה** - שמרו את השינויים

### 3. בדיקה

1. **הרצת המשחק** - בדקו את השינויים על ידי הרצת המשחק
2. **תיקון שגיאות** - אם יש בעיות, חזרו לקובץ ותקנו אותן
3. **חזרה על התהליך** - חזרו על שלבי העריכה והבדיקה עד שתהיו מרוצים מהתוצאה

## עריכת סוגי תוכן ספציפיים

### עריכת הרפתקאות (סיפור ודיאלוגים)

הרפתקאות מאוחסנות בקבצי JSON בתיקייה `src/data/adventures`. כל קובץ מייצג פרק או חלק מהסיפור.

```json
{
  "title": "ימי הכבוד",
  "description": "שנות שפע במצרים, בני ישראל זוכים ליחס של כבוד.",
  "scenes": [
    {
      "speaker": "מספר",
      "text": "שנות שפע במצרים. השמש מאירה על ערים משגשגות...",
      "background_img": "/images/adventures/days_of_honor/prosperity_1.png"
    }
  ]
}
```

למדריך מפורט על עריכת הרפתקאות, ראו [מדריך לעריכת הרפתקאות](/docs/adventure-editing).

### עריכת טריוויה

שאלות טריוויה מאוחסנות בקבצי JSON בתיקייה `src/data/hebrew-trivia`. כל קובץ מכיל אוסף שאלות בנושא מסוים.

```json
{
  "title_he": "ימים ראשונים במצרים",
  "description_he": "שאלות על תחילת דרכם של בני ישראל במצרים...",
  "questions": [
    {
      "question_he": "כיצד מתואר יחס המצרים אל בני ישראל בתקופה הראשונה?",
      "correct_answer_he": "טוב, מחמיא, מתחשב ומכבד",
      "incorrect_answers_he": [
        "חשדני ועוין מלכתחילה",
        "אדיש ולא מתעניין בהם כלל",
        "מתנשא ומזלזל"
      ]
    }
  ]
}
```

למדריך מפורט על עריכת טריוויה, ראו [מדריך לעריכת טריוויה](/docs/trivia-editing).

### ניהול נכסים חזותיים

הנכסים החזותיים (תמונות) מאוחסנים בתיקיית `public`. כדי להחליף תמונה קיימת, פשוט החליפו את הקובץ בגרסה החדשה (שמרו על אותו שם קובץ).

למדריך מפורט על ניהול נכסים, ראו [ניהול תמונות ונכסים](/docs/asset-management).

### עריכת מיני-משחקים

המיני-משחקים השונים מוגדרים בקוד, אך חלקם משתמשים בנתונים שניתן לערוך:

- **טריוויה** - ראו [מדריך לעריכת טריוויה](/docs/trivia-editing)
- **ניהול משימות יומי** - ראו [מנהל משימות יומי](/docs/daily-task-manager)
- **Space Invaders** - ראו [Space Invaders](/docs/space-invaders)

## טיפים לעריכת תוכן

1. **שמרו על עקביות** - וודאו שהתוכן החדש מתאים לסגנון ולטון של שאר המשחק
2. **בדקו היטב** - תמיד בדקו את השינויים שלכם במשחק עצמו
3. **גבו את העבודה שלכם** - שמרו עותקים של הקבצים המקוריים והמעודכנים
4. **תעדו את השינויים** - נהלו רשימה של השינויים שביצעתם
5. **התחילו קטן** - אם אתם חדשים בעריכת תוכן, התחילו בשינויים קטנים וצרו בהדרגה

## שגיאות נפוצות ופתרונן

### שגיאות JSON

אם המשחק מפסיק לעבוד אחרי עריכת קובץ JSON, ייתכן שיש שגיאת תחביר:

- **פסיקים חסרים או מיותרים** - וודאו שיש פסיק אחרי כל זוג מפתח-ערך, חוץ מהאחרון בכל אובייקט
- **מרכאות חסרות** - וודאו שכל המחרוזות מוקפות במרכאות כפולות (`"`)
- **סוגריים לא סגורים** - וודאו שכל סוגר פותח (`{` או `[`) יש סוגר סוגר מתאים (`}` או `]`)

### שגיאות נתיבי קבצים

אם תמונות לא מוצגות:
- וודאו שנתיב הקובץ נכון (למשל, `/images/adventures/...`)
- וודאו שהקובץ אכן קיים בנתיב המצוין
- וודאו שאותיות גדולות/קטנות תואמות (חשוב במיוחד בשרתי Linux)

## סיכום

עריכת תוכן במשחק "יציאת מצרים" היא תהליך פשוט יחסית, גם למשתמשים ללא רקע טכני. על ידי הבנת מבנה הקבצים ועקרונות העריכה הבסיסיים, תוכלו להתאים את המשחק לצרכים שלכם.

במדריכים הבאים, נעמיק בכל אחד מסוגי התוכן ונלמד כיצד לערוך אותם בצורה מפורטת יותר.
