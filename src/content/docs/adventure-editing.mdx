# מדריך לעריכת הרפתקאות

מדריך זה יסביר כיצד לערוך ולהוסיף תוכן סיפורי למשחק "יציאת מצרים" באמצעות קבצי JSON. המדריך מיועד למשתמשים ללא רקע טכני, ויסביר את התהליך בצורה פשוטה וברורה.

## מהם קבצי הרפתקאות?

קבצי ההרפתקאות הם קבצי JSON המכילים את הסיפור, הדיאלוגים, הבחירות והתמונות של כל פרק במשחק. הם נמצאים בתיקייה `src/data/adventures` ושמותיהם מתארים את תוכנם (לדוגמה, `days_of_honor.json` עבור "ימי הכבוד").

## מבנה בסיסי של קובץ הרפתקה

כל קובץ הרפתקה מורכב משלושה חלקים עיקריים:

1. **כותרת ותיאור** - מידע כללי על ההרפתקה
2. **סצנות** - רצף של רגעים בסיפור, כולל דיאלוגים ותיאורים
3. **בחירות** - אפשרויות שהשחקן יכול לבחור בהן, המשפיעות על התפתחות הסיפור

להלן דוגמה פשוטה של קובץ הרפתקה:

```json
{
  "title": "ימי הכבוד",
  "description": "שנות שפע במצרים, בני ישראל זוכים ליחס של כבוד.",
  "scenes": [
    {
      "speaker": "מספר",
      "text": "שנות שפע במצרים. השמש מאירה על ערים משגשגות...",
      "character_left_img": "/characters/ohad/image.png",
      "character_right_img": null,
      "background_img": "/images/adventures/days_of_honor/prosperity_1.png"
    },
    // סצנות נוספות...
  ]
}
```

## הסבר מפורט על השדות

### שדות ברמה העליונה

* **`title`** (מחרוזת): הכותרת הראשית של ההרפתקה (לדוגמה, "ימי הכבוד").
* **`description`** (מחרוזת): תיאור קצר של נושא ההרפתקה (לדוגמה, "שנות שפע במצרים, בני ישראל זוכים ליחס של כבוד.").
* **`scenes`** (מערך של אובייקטי סצנה): זהו הליבה של ההרפתקה, המכילה רשימה מסודרת של סצנות המרכיבות את זרימת הסיפור.

### שדות של אובייקט סצנה

כל אובייקט במערך `scenes` מייצג צעד או רגע בודד בהרפתקה.

* **`id`** (מחרוזת, אופציונלי אך מומלץ לסצנות שמקושרות מבחירות): מזהה ייחודי לסצנה. חשוב במיוחד לניווט באמצעות `choices` ו-`next`. הסצנה הראשונה בדרך כלל לא צריכה `id` אלא אם מגיעים אליה מסצנה אחרת. מומלץ להשתמש בשמות תיאוריים (לדוגמה, `start_honor`, `accept_eagerly`).
* **`speaker`** (מחרוזת): שם הדמות המדברת או "מספר" עבור טקסט נרטיבי.
* **`text`** (מחרוזת): הדיאלוג או הטקסט הנרטיבי שיוצג בסצנה זו.
* **`character_left_img`** (מחרוזת או `null`): הנתיב לקובץ התמונה של הדמות שתוצג בצד שמאל של המסך. השתמשו בנתיב יחסי המתחיל מ-`/public/`. אם `null`, לא תוצג דמות בצד שמאל.
* **`character_right_img`** (מחרוזת או `null`): הנתיב לקובץ התמונה של הדמות שתוצג בצד ימין של המסך. השתמשו בנתיב יחסי המתחיל מ-`/public/`. אם `null`, לא תוצג דמות בצד ימין.
* **`background_img`** (מחרוזת): הנתיב לתמונת הרקע של הסצנה. השתמשו בנתיב יחסי המתחיל מ-`/public/`.
* **`choices`** (מערך של אובייקטי בחירה, אופציונלי): אם קיים, שדה זה מספק לשחקן אפשרויות לבחירה. ההרפתקה נעצרת כאן עד שהשחקן בוחר. אם הושמט, ההרפתקה ממשיכה אוטומטית לסצנה הבאה במערך או לסצנה שצוינה ב-`next`.
* **`next`** (מחרוזת, אופציונלי): ה-`id` של הסצנה שאליה יש לקפוץ *אחרי* שסצנה זו מוצגת. משמש לקידום לינארי *אחרי* סצנה ללא בחירה או להכוונת הזרימה אחרי שבחירה הובילה לסצנת תוצאה ספציפית. אם גם `choices` וגם `next` חסרים, המשחק ממשיך לסצנה הבאה ברצף במערך `scenes`.
* **`end`** (בוליאני, אופציונלי): אם מוגדר כ-`true`, זה מסמן את סוף פרק ההרפתקה אחרי שסצנה זו מוצגת.

### שדות של אובייקט בחירה

כל אובייקט במערך `choices` מייצג אפשרות בודדת המוצגת לשחקן.

* **`text`** (מחרוזת): הטקסט המוצג על כפתור הבחירה לשחקן (לדוגמה, "הסכם בהתלהבות: \"בכבוד רב! אשמח לסייע בכל דרך.\"").
* **`outcome`** (מחרוזת): ה-`id` של הסצנה שאליה יש לנווט כאשר בחירה זו נבחרת. ה-`id` הזה חייב להתאים לשדה `id` של אובייקט סצנה אחר במערך `scenes`.

## זרימת הסיפור

1. **התחלה:** ההרפתקה מתחילה עם אובייקט הסצנה הראשון במערך `scenes`.
2. **התקדמות:**
   * אם לסצנה אין `choices` ואין `next`, ההרפתקה ממשיכה ל*אובייקט הסצנה הבא* במערך.
   * אם לסצנה יש שדה `next`, ההרפתקה קופצת לסצנה שה-`id` שלה תואם את ערך ה-`next`.
   * אם לסצנה יש מערך `choices`, ההרפתקה נעצרת, מציגה את הבחירות, ומחכה לקלט מהשחקן.
3. **בחירות:** כאשר השחקן בוחר באפשרות, ההרפתקה קופצת לסצנה שה-`id` שלה תואם את ערך ה-`outcome` של האפשרות שנבחרה.
4. **סיום:** ההרפתקה מסתיימת כאשר מגיעים לסצנה עם `"end": true`.

## כיצד לערוך קובץ הרפתקה קיים

### 1. פתיחת הקובץ

פתחו את קובץ ההרפתקה שברצונכם לערוך בעורך טקסט (מומלץ Visual Studio Code). הקבצים נמצאים בתיקייה `src/data/adventures`.

### 2. עריכת טקסט

כדי לערוך טקסט קיים, פשוט שנו את הערך של שדה ה-`text` בסצנה הרלוונטית:

```json
{
  "speaker": "מספר",
  "text": "זהו הטקסט החדש שיוצג במשחק.", // שינוי הטקסט כאן
  "character_left_img": "/characters/ohad/image.png",
  "character_right_img": null,
  "background_img": "/images/adventures/days_of_honor/prosperity_1.png"
}
```

### 3. שינוי דמויות או רקעים

כדי להחליף תמונת דמות או רקע, שנו את הנתיב בשדות המתאימים:

```json
{
  "speaker": "מספר",
  "text": "שנות שפע במצרים. השמש מאירה על ערים משגשגות...",
  "character_left_img": "/characters/ohad/new_image.png", // שינוי תמונת הדמות
  "character_right_img": null,
  "background_img": "/images/adventures/days_of_honor/new_background.png" // שינוי תמונת הרקע
}
```

### 4. הוספת סצנה חדשה

כדי להוסיף סצנה חדשה, הוסיפו אובייקט חדש למערך `scenes`:

```json
{
  "scenes": [
    // סצנות קיימות...
    {
      "speaker": "אוהד",
      "text": "זוהי סצנה חדשה שהוספתי למשחק.",
      "character_left_img": "/characters/ohad/image.png",
      "character_right_img": null,
      "background_img": "/images/adventures/days_of_honor/prosperity_1.png"
    }
  ]
}
```

### 5. הוספת בחירה

כדי להוסיף בחירה לסצנה, הוסיפו שדה `choices` עם האפשרויות השונות:

```json
{
  "speaker": "מספר",
  "text": "מה תרצה לעשות עכשיו?",
  "character_left_img": "/characters/ohad/image.png",
  "character_right_img": null,
  "background_img": "/images/adventures/days_of_honor/prosperity_1.png",
  "choices": [
    { "text": "ללכת לשוק", "outcome": "go_to_market" },
    { "text": "להישאר בבית", "outcome": "stay_home" }
  ]
}
```

זכרו להוסיף גם את הסצנות המתאימות עם ה-`id` המתאים:

```json
{
  "id": "go_to_market",
  "speaker": "מספר",
  "text": "החלטת ללכת לשוק...",
  // שאר השדות...
}
```

## כיצד ליצור קובץ הרפתקה חדש

### 1. יצירת קובץ חדש

צרו קובץ JSON חדש בתיקייה `src/data/adventures` עם שם מתאים (לדוגמה, `new_adventure.json`).

### 2. הגדרת המבנה הבסיסי

העתיקו את המבנה הבסיסי הבא והתאימו אותו לצרכים שלכם:

```json
{
  "title": "כותרת ההרפתקה החדשה",
  "description": "תיאור קצר של ההרפתקה החדשה.",
  "scenes": [
    {
      "speaker": "מספר",
      "text": "טקסט הפתיחה של ההרפתקה החדשה.",
      "character_left_img": "/characters/ohad/image.png",
      "character_right_img": null,
      "background_img": "/images/adventures/new_adventure/background.png"
    }
    // הוסיפו סצנות נוספות כאן...
  ]
}
```

### 3. הוספת סצנות, בחירות ותוצאות

הוסיפו סצנות נוספות למערך `scenes`, כולל בחירות ותוצאות כפי שהוסבר לעיל.

## טיפים ושיטות עבודה מומלצות

### תכנון מראש

לפני שמתחילים לערוך או ליצור הרפתקה, מומלץ לתכנן את מבנה הסיפור והבחירות מראש. אפשר להשתמש בתרשים זרימה פשוט:

```
התחלה
  |
  +--> בחירה 1 --> תוצאה 1 --> המשך משותף
  |                                |
  +--> בחירה 2 --> תוצאה 2 --------+
                                    |
                                  סיום
```

### שמירה על עקביות

* השתמשו באותן דמויות ורקעים באופן עקבי לאורך ההרפתקה
* שמרו על סגנון כתיבה אחיד
* וודאו שהסיפור מתחבר היטב להרפתקאות אחרות במשחק

### בדיקת תקינות ה-JSON

לפני שמשלימים את העריכה, וודאו שקובץ ה-JSON תקין:
* כל המרכאות הכפולות (`"`) סגורות כראוי
* יש פסיק (`,`) אחרי כל זוג מפתח-ערך, חוץ מהאחרון בכל אובייקט
* כל הסוגריים המסולסלים (`{` ו-`}`) והמרובעים (`[` ו-`]`) סגורים כראוי

### בדיקת הקישורים

וודאו שכל ערכי ה-`outcome` מצביעים על `id` קיים, וכל ערכי ה-`next` מצביעים על `id` קיים.

## סיכום

עריכת ויצירת הרפתקאות במשחק "יציאת מצרים" היא תהליך פשוט יחסית, גם למשתמשים ללא רקע טכני. על ידי הבנת מבנה קובץ ה-JSON ועקרונות הזרימה של הסיפור, תוכלו ליצור חוויות מרתקות ואינטראקטיביות.

זכרו לתכנן את הסיפור מראש, לשמור על עקביות, ולבדוק את הקובץ לפני שמשלימים את העריכה. בהצלחה!
