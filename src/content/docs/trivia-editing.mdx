# מדריך לעריכת טריוויה

מדריך זה יסביר כיצד לערוך ולהוסיף שאלות טריוויה למשחק "יציאת מצרים" באמצעות קבצי JSON. המדריך מיועד למשתמשים ללא רקע טכני, ויסביר את התהליך בצורה פשוטה וברורה.

## מהם קבצי הטריוויה?

קבצי הטריוויה הם קבצי JSON המכילים את השאלות, התשובות הנכונות, התשובות השגויות וההסברים של מיני-משחק הטריוויה. הם נמצאים בתיקייה `src/data/hebrew-trivia` ומחולקים לפי נושאים:

- `early-days.json` - שאלות על תחילת דרכם של בני ישראל במצרים
- `slavery.json` - שאלות על תקופת השעבוד
- `plagues.json` - שאלות על המכות
- `transition.json` - שאלות על המעבר והיציאה ממצרים

## מבנה בסיסי של קובץ טריוויה

כל קובץ טריוויה מורכב משלושה חלקים עיקריים:

1. **כותרת ותיאור** - מידע כללי על נושא הטריוויה
2. **שאלות** - מערך של שאלות בנושא
3. **לכל שאלה** - תשובה נכונה, תשובות שגויות והסבר

להלן דוגמה פשוטה של קובץ טריוויה:

```json
{
  "title_he": "ימים ראשונים במצרים",
  "description_he": "שאלות על תחילת דרכם של בני ישראל במצרים, מעמדם המכובד והיחס הטוב שקיבלו.",
  "questions": [
    {
      "question_he": "כיצד מתואר יחס המצרים אל בני ישראל בתקופה הראשונה לשהותם בארץ?",
      "correct_answer_he": "טוב, מחמיא, מתחשב ומכבד",
      "incorrect_answers_he": [
        "חשדני ועוין מלכתחילה",
        "אדיש ולא מתעניין בהם כלל",
        "מתנשא ומזלזל"
      ],
      "explanation_he": "הטקסט מציין: \"בתקופה הראשונה לשהותינו בארץ מצרים היחס אלינו היה טוב, מחמיא, מתחשב ומכבד.\""
    },
    // שאלות נוספות...
  ]
}
```

## הסבר מפורט על השדות

### שדות ברמה העליונה

* **`title_he`** (מחרוזת): הכותרת הראשית של נושא הטריוויה בעברית (לדוגמה, "ימים ראשונים במצרים").
* **`description_he`** (מחרוזת): תיאור קצר של נושא הטריוויה בעברית (לדוגמה, "שאלות על תחילת דרכם של בני ישראל במצרים...").
* **`questions`** (מערך של אובייקטי שאלה): מערך המכיל את כל השאלות בנושא זה.

### שדות של אובייקט שאלה

כל אובייקט במערך `questions` מייצג שאלת טריוויה אחת:

* **`question_he`** (מחרוזת): טקסט השאלה בעברית.
* **`correct_answer_he`** (מחרוזת): התשובה הנכונה לשאלה בעברית.
* **`incorrect_answers_he`** (מערך של מחרוזות): מערך של תשובות שגויות בעברית. בדרך כלל יש 3 תשובות שגויות לכל שאלה.
* **`explanation_he`** (מחרוזת): הסבר קצר לתשובה הנכונה בעברית. זה יוצג לשחקן לאחר שבחר תשובה.

## כיצד לערוך קובץ טריוויה קיים

### 1. פתיחת הקובץ

פתחו את קובץ הטריוויה שברצונכם לערוך בעורך טקסט (מומלץ Visual Studio Code). הקבצים נמצאים בתיקייה `src/data/hebrew-trivia`.

### 2. עריכת שאלה קיימת

כדי לערוך שאלה קיימת, מצאו את אובייקט השאלה הרלוונטי במערך `questions` ושנו את הערכים הרצויים:

```json
{
  "question_he": "זוהי השאלה החדשה?", // שינוי טקסט השאלה
  "correct_answer_he": "זוהי התשובה הנכונה החדשה", // שינוי התשובה הנכונה
  "incorrect_answers_he": [
    "תשובה שגויה חדשה 1", // שינוי תשובות שגויות
    "תשובה שגויה חדשה 2",
    "תשובה שגויה חדשה 3"
  ],
  "explanation_he": "זהו ההסבר החדש לתשובה הנכונה." // שינוי ההסבר
}
```

### 3. הוספת שאלה חדשה

כדי להוסיף שאלה חדשה, הוסיפו אובייקט שאלה חדש למערך `questions`:

```json
{
  "questions": [
    // שאלות קיימות...
    {
      "question_he": "זוהי שאלה חדשה שהוספתי?",
      "correct_answer_he": "זוהי התשובה הנכונה",
      "incorrect_answers_he": [
        "תשובה שגויה 1",
        "תשובה שגויה 2",
        "תשובה שגויה 3"
      ],
      "explanation_he": "זהו הסבר לתשובה הנכונה."
    }
  ]
}
```

### 4. מחיקת שאלה

כדי למחוק שאלה, פשוט מחקו את כל אובייקט השאלה מהמערך `questions`. וודאו שהפסיקים (`,`) נשארים תקינים - צריך להיות פסיק אחרי כל שאלה חוץ מהאחרונה.

## כיצד ליצור קובץ טריוויה חדש

### 1. יצירת קובץ חדש

צרו קובץ JSON חדש בתיקייה `src/data/hebrew-trivia` עם שם מתאים (לדוגמה, `new_topic.json`).

### 2. הגדרת המבנה הבסיסי

העתיקו את המבנה הבסיסי הבא והתאימו אותו לצרכים שלכם:

```json
{
  "title_he": "כותרת הנושא החדש",
  "description_he": "תיאור קצר של נושא הטריוויה החדש.",
  "questions": [
    {
      "question_he": "זוהי השאלה הראשונה?",
      "correct_answer_he": "זוהי התשובה הנכונה",
      "incorrect_answers_he": [
        "תשובה שגויה 1",
        "תשובה שגויה 2",
        "תשובה שגויה 3"
      ],
      "explanation_he": "זהו הסבר לתשובה הנכונה."
    }
    // הוסיפו שאלות נוספות כאן...
  ]
}
```

### 3. הוספת שאלות נוספות

הוסיפו שאלות נוספות למערך `questions` כפי שהוסבר לעיל.

## טיפים ליצירת שאלות טריוויה איכותיות

### 1. רמת קושי מאוזנת

* שלבו שאלות ברמות קושי שונות - קלות, בינוניות וקשות
* התחילו עם שאלות קלות יותר והעלו את רמת הקושי בהדרגה

### 2. תשובות ברורות וחד-משמעיות

* וודאו שיש רק תשובה נכונה אחת ברורה
* הימנעו משאלות שיש להן יותר מתשובה אפשרית אחת
* וודאו שהתשובות השגויות הגיוניות אך בבירור לא נכונות

### 3. תשובות שגויות משכנעות

* צרו תשובות שגויות שנשמעות אמינות אך שגויות
* הימנעו מתשובות שגויות מגוחכות או שקל מדי לפסול
* נסו ליצור "מסיחים" שמבוססים על טעויות נפוצות או בלבול

### 4. הסברים מועילים

* ספקו הסברים קצרים וברורים לתשובות הנכונות
* הוסיפו מקור או ציטוט כשרלוונטי
* השתמשו בהסברים כהזדמנות ללמידה נוספת

### 5. איזון בין סוגי שאלות

* שלבו שאלות עובדתיות (מי, מה, מתי, היכן)
* הוסיפו שאלות הבנה (למה, כיצד)
* שקלו להוסיף שאלות השוואה או ניתוח

## בדיקת תקינות הטריוויה

### 1. בדיקת תחביר ה-JSON

לפני שמשלימים את העריכה, וודאו שקובץ ה-JSON תקין:
* כל המרכאות הכפולות (`"`) סגורות כראוי
* יש פסיק (`,`) אחרי כל זוג מפתח-ערך וכל פריט במערך, חוץ מהאחרון
* כל הסוגריים המסולסלים (`{` ו-`}`) והמרובעים (`[` ו-`]`) סגורים כראוי

### 2. בדיקת תוכן

* וודאו שאין שגיאות כתיב או דקדוק
* וודאו שהתשובה הנכונה אכן נכונה
* וודאו שאין כפילויות בין התשובות הנכונות והשגויות
* וודאו שההסבר מתאים לתשובה הנכונה

## סיכום

עריכת ויצירת שאלות טריוויה במשחק "יציאת מצרים" היא תהליך פשוט יחסית, גם למשתמשים ללא רקע טכני. על ידי הבנת מבנה קובץ ה-JSON ועקרונות יצירת שאלות טובות, תוכלו להעשיר את חווית המשחק עם תוכן חינוכי ומעניין.

זכרו ליצור שאלות מגוונות ברמות קושי שונות, עם תשובות ברורות והסברים מועילים. בדקו תמיד את תקינות הקובץ לפני שמשלימים את העריכה. בהצלחה!
