# Content Editing Guide

This guide explains how to edit content within the Exodus game project.

## Editing Process

1.  Locate the relevant content file (e.g., `.json`, `.mdx`).
2.  Make the necessary changes using a text editor or CMS interface.
3.  Save the file.
4.  Commit the changes to version control (if applicable).

## Example: Editing Dialogue

Dialogue might be stored in JSON files like this:

```json
{
  "npc_01": {
    "greeting": "Welcome, traveler!",
    "farewell": "Safe journeys."
  }
}
```

Simply modify the text values as needed.
